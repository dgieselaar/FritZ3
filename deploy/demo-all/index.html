This is a crude demo showcasing the potential of our FritZ3 framework.
If you change the values (stylesheet/id/className/name) the display automatically updates.
It's obviously still buggy, and I haven't yet implemented the vertical layout.

Please note that the css parser is very easy to break.
The Github repo is currently private but we're hoping to open-source a lot of the code in the next few months.
